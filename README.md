# vfr2cfr
超簡単にvfr動画をcfr動画に変換する。

## つかいかた
`開く` で動画を開く。  
真ん中のボタンでフレームレートを設定。  
`変換` で変換する。  

## 前提
* ffmpeg
* Ut Video Codec Suite

## つけたい機能
* ドラッグアンドドロップでファイル追加。

## できたこと
* ffmpegの（標準？）出力をひろう  
何とかして拾って、何%エンコードできたか表示したい。  
たぶんffprobeとかで元動画のフレーム数とれるはずだから、それをエンコードしたフレーム数で割ればいいと思う。  
	* 非同期処理とタイマーでなんとかした。  
* statusStripでなに変換してるかくらいは表示する。  
	* 表示できた。
* TextBoxはオタクっぽいので消す。
	* 残すことにした。
* レイアウト調整
	* した。
* フレームレート設定機能。30と60だけでok。  
	* した。
* 中断したとき。  
	* エンコード中に閉じたら警告出すようにした。
	* それでも閉じたときは無理やりプロセス止めるようにした。
